│ APRÈS (Nouvelles Cartes Produits avec Sélecteur)                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  ┌─────────────────────────────────┐                                         │
│  │  [IMAGE DU PRODUIT]             │                                         │
│  ├─────────────────────────────────┤                                         │
│  │ Titre du Produit                │                                         │
│  │ Auteur                          │                                         │
│  │ 6.90 €                          │                                         │
│  │                                 │                                         │
│  │ ┌──────────────┐  ┌────────────┐│                                         │
│  │ │ − │  1  │ +  │  │ AJOUTER  ││ ← Nouvel UI avec sélecteur + bouton     │
│  │ └──────────────┘  └────────────┘│                                         │
│  └─────────────────────────────────┘                                         │
│                                                                               │
│  INTERACTIONS:                                                               │
│  - Cliquer "-" : Diminue (min 1)        → Affiche 1, 2, 3... jusqu'à 99    │
│  - Voir chiffre : Quantité sélectionnée → Permet édition manuelle (1-99)   │
│  - Cliquer "+" : Augmente (max 99)      → Saute de 1 à la fois             │
│  - Cliquer "Ajouter" : Ajoute quantité sélectionnée au panier              │
│                                                                               │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ EXEMPLE D'UTILISATION                                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│ 1. Utilisateur voit: [ - │ 1 │ + ]                    [ AJOUTER ]          │
│                                                                               │
│ 2. Clique 4 fois sur "+":                                                   │
│    [ - │ 1 │ + ] → [ - │ 2 │ + ] → [ - │ 3 │ + ] → [ - │ 4 │ + ] → ...    │
│                                                                               │
│ 3. Résultat: [ - │ 5 │ + ]                            [ AJOUTER ]          │
│                                                                               │
│ 4. Clique "AJOUTER" → Notification: "5x [Produit] ajouté au panier!"       │
│                                                                               │
│ 5. Automatiquement réinitialisé à: [ - │ 1 │ + ]                           │
│                                                                               │
└─────────────────────────────────────────────────────────────────────────────┘
